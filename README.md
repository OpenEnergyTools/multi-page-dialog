# \<multi-page-dialog>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation

```bash
npm i multi-page-dialog
```

## Usage

```html
<script type="module">
  import 'multi-page-dialog';
</script>

<multi-page-dialog></multi-page-dialog>
```
